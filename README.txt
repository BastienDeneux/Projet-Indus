# Projet-Indus

Afin de faire fonctionner le projet sur marchine, il suffit de build un projet ionic et dans le dossier www créé il faut :
  -remplacer le fichier index.html par le nôtre
  -remplacer le fichier app.js par le nôtre
  -ajouter le répertoire pages contenant les fichiers home.html et fiche.html
  
  et puis lancer un ionic serve dans l'invite de commande (en étant placé dans le dossier de l'application)
